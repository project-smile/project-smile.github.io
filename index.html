Project Smile - Wat heeft jou vandaag doen glimlachen?

Project Smile is een initatief om mensen op een ludieke manier te laten glimlachen.

Hoe werkt het?

Project Smile geeft regelmatig, verspreid over Nederland, een aantal kaartjes weg. Deze kaartjes, ter grootte van een visitekaart kaart, bevatten een bericht in de vorm van een grap, compliment of een opdracht. Het doel is om

De bedoeling is dat deze kaartjes steeds doorgegeven worden. Ontvang je een kaartje? Geef hem dan dezelfde of de volgende dag nog door aan iemand anders. Hoe meer mensen het kaartje te zien krijgen hoe beter!

Op het kaartje staat een link naar een unieke pagina. Wanneer je naar die link gaat zul je je kaartje digitaal zien zodat je hem ook via Facebook of Twitter kunt delen met je vrienden.
